📋 PASO 2: CONFIGURAR MERCADOPAGO (Opcional - 5 minutos)
---------------------------------------------------------

1. Ve a: https://www.mercadopago.com.co/developers
2. Crea una aplicación
3. Copia tu Access Token
4. Agrégalo en .env:

MERCADOPAGO_ACCESS_TOKEN=tu_token_aqui
MERCADOPAGO_PUBLIC_KEY=tu_public_key_aqui


📋 PASO 3: CONFIGURAR PAYPAL (Opcional - 5 minutos)
----------------------------------------------------

1. Ve a: https://developer.paypal.com
2. Crea una aplicación
3. Copia Client ID y Secret
4. Agrégalos en .env:

PAYPAL_CLIENT_ID=tu_client_id_aqui
PAYPAL_CLIENT_SECRET=tu_secret_aqui


📋 PASO 4: PROBAR EL SISTEMA (1 minuto)
----------------------------------------

npx tsx scripts/test-payment-links.ts

Esto mostrará:
✅ Links generados para tus productos
✅ Instrucciones para WhatsApp
✅ Respuestas según método elegido


📋 PASO 5: INICIAR EL SERVIDOR (1 minuto)
------------------------------------------

npm run dev

Luego:
1. Abre http://localhost:3000
2. Verás el botón verde flotante (Asistente)
3. Haz clic y pregunta: "Cómo configurar pagos?"


📋 PASO 6: PROBAR EN WHATSAPP (2 minutos)
------------------------------------------

1. Conecta WhatsApp (escanea QR)
2. Envía desde tu teléfono:
   - "Info del curso de piano"
   - "Dame el link de pago"
3. Verás todas las opciones de pago
