📋 PRÓXIMOS PASOS PARA DEPLOY EN EASYPANEL

1. CONECTAR EASYPANEL A GITHUB
   - Ve a tu app en Easypanel
   - Conecta el repositorio: daveymena/bot-intelogente
   - Branch: main
   - Easypanel hará pull automáticamente

2. CONFIGURAR VARIABLES DE ENTORNO
   - Copia las variables de tu .env local
   - Agrega en Easypanel (ver VARIABLES_EASYPANEL_FINALES.txt)
   - Importante: Cambiar DATABASE_URL, NEXTAUTH_SECRET, etc.

3. DEPLOY
   - Easypanel ejecutará:
     * npm install
     * npm run build
     * npm start

4. POST-DEPLOY (En terminal de Easypanel)
   npm run db:push
   npm run db:generate
   npx tsx scripts/create-admin-production.ts

5. CONFIGURAR WEBHOOKS
   - MercadoPago: https://tudominio.com/api/payments/webhook
   - PayPal: https://tudominio.com/api/payments/webhook
   - Ver guía: CONFIGURAR_WEBHOOKS_FACIL.md
