VARIABLES DE ENTORNO PARA EASYPANEL
═══════════════════════════════════════════════════════════════════

COPIA Y PEGA ESTAS VARIABLES EN EASYPANEL:

───────────────────────────────────────────────────────────────────
1. IA - CRÍTICO (Sin esto el bot no funciona)
───────────────────────────────────────────────────────────────────

GROQ_API_KEY=gsk_tu_key_aqui

Dónde conseguirla:
→ https://console.groq.com/keys
→ Crear cuenta gratis
→ Generar API key


───────────────────────────────────────────────────────────────────
2. BASE DE DATOS - CRÍTICO
───────────────────────────────────────────────────────────────────

DATABASE_URL=postgresql://usuario:password@host:5432/database

Opciones:

A) Usar PostgreSQL de Easypanel:
   1. Crear servicio PostgreSQL en Easypanel
   2. Copiar la URL que te da
   3. Pegar aquí

B) Usar tu base de datos actual:
   postgresql://neondb_owner:tu_password@ep-host.region.aws.neon.tech/neondb?sslmode=require


───────────────────────────────────────────────────────────────────
3. AUTENTICACIÓN - CRÍTICO
───────────────────────────────────────────────────────────────────

NEXTAUTH_SECRET=genera_un_string_aleatorio_largo_minimo_32_caracteres

Generar con:
→ https://generate-secret.vercel.app/32
→ O ejecutar: node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"

NEXTAUTH_URL=https://tu-app.easypanel.app

Reemplazar con tu URL de Easypanel


───────────────────────────────────────────────────────────────────
4. EMAIL - IMPORTANTE (Para notificaciones)
───────────────────────────────────────────────────────────────────

RESEND_API_KEY=re_tu_key_aqui

Dónde conseguirla:
→ https://resend.com/api-keys
→ Crear cuenta gratis
→ Generar API key


───────────────────────────────────────────────────────────────────
5. PAGOS - IMPORTANTE (Para procesar pagos)
───────────────────────────────────────────────────────────────────

PAYPAL_CLIENT_ID=tu_client_id_aqui
PAYPAL_CLIENT_SECRET=tu_secret_aqui

Dónde conseguirlas:
→ https://developer.paypal.com/dashboard/
→ Crear app
→ Copiar Client ID y Secret


───────────────────────────────────────────────────────────────────
6. CONFIGURACIÓN GENERAL
───────────────────────────────────────────────────────────────────

NODE_ENV=production
PORT=3000


───────────────────────────────────────────────────────────────────
7. OPCIONAL (Mejoras futuras)
───────────────────────────────────────────────────────────────────

# MercadoPago (si quieres agregarlo después)
MERCADOPAGO_ACCESS_TOKEN=tu_token_aqui
MERCADOPAGO_PUBLIC_KEY=tu_public_key_aqui

# Ollama (para IA local)
OLLAMA_BASE_URL=http://localhost:11434
OLLAMA_MODEL=llama3.1

═══════════════════════════════════════════════════════════════════
CÓMO AGREGAR EN EASYPANEL
═══════════════════════════════════════════════════════════════════

1. Ir a tu app en Easypanel
2. Click en "Environment"
3. Click en "Add Variable"
4. Copiar nombre (ej: GROQ_API_KEY)
5. Pegar valor
6. Repetir para cada variable
7. Click en "Save"
8. Reiniciar app


═══════════════════════════════════════════════════════════════════
VERIFICAR VARIABLES
═══════════════════════════════════════════════════════════════════

Después de agregar, verifica en logs:

✅ "GROQ_API_KEY loaded"
✅ "Database connected"
✅ "NextAuth configured"

Si ves errores:
❌ "GROQ_API_KEY not found" → Agregar variable
❌ "Database connection failed" → Verificar DATABASE_URL
❌ "NextAuth error" → Verificar NEXTAUTH_SECRET
