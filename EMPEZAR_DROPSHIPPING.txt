🛍️ SISTEMA DE DROPSHIPPING - SMARTJOYS
========================================

¡Listo para importar productos automáticamente!

📋 COMANDOS RÁPIDOS:
-------------------

1️⃣ Scrapear productos de SmartJoys:
   npm run scrape:smartjoys

2️⃣ Importar a tu base de datos:
   npm run import:smartjoys

3️⃣ Hacer ambos (actualización completa):
   npm run dropship:update


🎯 FLUJO COMPLETO:
-----------------

1. Ejecuta: npm run dropship:update
2. Espera 2-3 minutos mientras se extraen los productos
3. Los productos aparecerán automáticamente en tu dashboard
4. Tu bot ya puede venderlos por WhatsApp
