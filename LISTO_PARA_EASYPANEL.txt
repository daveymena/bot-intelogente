🚀 PASOS PARA DEPLOY (5 MINUTOS)

1️⃣ CAMBIAR SCHEMA A POSTGRESQL

   Abrir: prisma/schema.prisma
   
   Cambiar:
   datasource db {
     provider = "sqlite"
     url      = env("DATABASE_URL")
   }
   
   Por:
   datasource db {
     provider = "postgresql"
     url      = env("DATABASE_URL")
   }

2️⃣ VERIFICAR PREPARACIÓN

   npx tsx scripts/preparar-produccion.ts

3️⃣ COMMIT Y PUSH

   git add .
   git commit -m "feat: Generador de personalidad v2.1 + PostgreSQL"
   git push origin main

4️⃣ CONFIGURAR EN EASYPANEL

   • Ir a tu app en Easypanel
   • Actualizar variables de entorno (ver DEPLOY_EASYPANEL_AHORA.md)
   • Esperar deploy automático

5️⃣ POST-DEPLOY

   En terminal de Easypanel:
   npm run db:push
   npm run db:generate
   npx tsx scripts/create-admin-production.ts
