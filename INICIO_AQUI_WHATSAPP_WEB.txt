🚀 WHATSAPP WEB ESTABLE - INICIO RÁPIDO

Meta bloqueó tu WhatsApp. Ahora usaremos WhatsApp Web (Baileys) de forma profesional.

═══════════════════════════════════════════════════════════════════════════════

📋 PASOS RÁPIDOS (30 minutos)

1️⃣ ACTUALIZAR .env
   Copiar estas variables:
   
   HEARTBEAT_INTERVAL=15000
   RECONNECT_ATTEMPTS_MAX=50
   RECONNECT_DELAY_BASE=1000
   RECONNECT_DELAY_MAX=30000
   ENABLE_CONNECTION_MONITOR=true
   CONNECTION_HEALTH_CHECK_INTERVAL=30000
   CONNECTION_HEALTH_THRESHOLD=80

2️⃣ EJECUTAR OPTIMIZACIÓN
   npx tsx scripts/optimizar-baileys-reconexion.ts

3️⃣ INSTALAR DEPENDENCIAS
   npm install

4️⃣ INICIAR BOT
   npm run dev

5️⃣ ESCANEAR QR
   - Abre http://localhost:4000
   - Ve a "Conexión WhatsApp"
   - Escanea QR con tu teléfono

6️⃣ VERIFICAR SALUD
   curl http://localhost:4000/api/whatsapp/health?userId=user123

🔧 TROUBLESHOOTING

QR no aparece:
  rm -rf ./auth_sessions/user123
  npm run dev

Desconexión frecuente:
  HEARTBEAT_INTERVAL=20000 npm run dev

Mensajes no se envían:
  curl http://localhost:4000/api/whatsapp/health?userId=user123

Bot no inicia:
  kill -9 $(lsof -t -i :4000)
  npm run dev
