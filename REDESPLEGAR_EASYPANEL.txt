════════════════════════════════════════════════════════════════
OPCIÓN 1: REDESPLEGAR DESDE EASYPANEL (MÁS FÁCIL)
════════════════════════════════════════════════════════════════

1. Entra a Easypanel
   └─ https://easypanel.io

2. Selecciona tu proyecto/aplicación
   └─ Busca tu bot en la lista

3. Busca una de estas secciones:
   ├─ "Deploy" o "Deployments"
   ├─ "Build" o "Builds"
   ├─ "Git" o "Source"
   └─ "Settings"

4. Busca y haz clic en uno de estos botones:
   ├─ "Redeploy" ⭐ (Este es el más común)
   ├─ "Rebuild"
   ├─ "Deploy Latest"
   ├─ "Trigger Deploy"
   └─ "Pull & Rebuild"

5. Espera 2-5 minutos mientras se construye

6. Limpia el cache de tu navegador:
   └─ Presiona: Ctrl + Shift + R (Windows)
   └─ O: Cmd + Shift + R (Mac)

7. ¡Verifica los cambios!

════════════════════════════════════════════════════════════════
OPCIÓN 2: VERIFICAR CONFIGURACIÓN DE GIT
════════════════════════════════════════════════════════════════

1. En Easypanel → Tu Proyecto → Settings o Source

2. Verifica que esté configurado así:
   ├─ Repository: daveymena/bot-intelogente
   ├─ Branch: main
   └─ Auto Deploy: ON (opcional)

3. Si está correcto, haz clic en "Redeploy"

════════════════════════════════════════════════════════════════
OPCIÓN 3: CONFIGURAR AUTO-DEPLOY (RECOMENDADO)
════════════════════════════════════════════════════════════════

Para que se actualice automáticamente con cada git push:

1. En Easypanel → Tu Proyecto → Settings
   └─ Busca "Auto Deploy" o "Automatic Deployments"
   └─ Actívalo (ON)

2. O configura Webhook:
   ├─ Copia la "Webhook URL" de Easypanel
   ├─ Ve a GitHub → Tu Repo → Settings → Webhooks
   ├─ Add webhook
   ├─ Pega la URL
   └─ Save

TROUBLESHOOTING
