```

2. Borrar variables actuales
3. Pegar: Ctrl+V
4. Save
```

### 3. Restart
```
1. Ir a General
2. Clic en Restart
3. Esperar 2-3 minutos
```

### 4. Verificar
```
1. Ir a Logs
2. Buscar: "[AI] 🧠 Usando sistema de razonamiento avanzado"
3. Probar el bot
```
