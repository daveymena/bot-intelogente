✅ SE AGREGÓ PANEL DE DEBUG

Ahora el componente mostrará información de debugging para
identificar por qué no se muestra el contenido.

═══════════════════════════════════════════════════════════════
  🚀 HAZ ESTO AHORA (2 PASOS)
═══════════════════════════════════════════════════════════════

  1️⃣  Refresca el navegador
      → Presiona Ctrl + Shift + R
      → O presiona F5

  2️⃣  Verifica el panel de debug
      → Verás un cuadro gris que dice:
        "Debug: Status = DISCONNECTED, QR = null, Loading = false"


═══════════════════════════════════════════════════════════════
  ✅ QUÉ DEBERÍAS VER
═══════════════════════════════════════════════════════════════

  Después de refrescar:

  ┌─────────────────────────────────────────┐
  │ Debug: Status = DISCONNECTED, QR = null │  ← Panel de debug
  └─────────────────────────────────────────┘

  ⚠️ Para conectar WhatsApp, necesitarás...

  Pasos para conectar:
  1. Haz clic en "Conectar WhatsApp"
  2. Aparecerá un código QR...

  ┌─────────────────────────────────────────┐
  │  [Conectar WhatsApp]                    │  ← Botón verde
  │  [Simular Conexión (Demo)]             │  ← Botón outline
  └─────────────────────────────────────────┘


═══════════════════════════════════════════════════════════════
  🐛 SI AÚN NO VES EL BOTÓN
═══════════════════════════════════════════════════════════════

  1. Abre DevTools (F12)
  2. Ve a la pestaña "Console"
  3. Busca errores en rojo
  4. Copia el error y compártelo

  O

  1. Abre en nueva pestaña:
     http://localhost:3000/api/whatsapp/status
  2. Copia la respuesta JSON
  3. Compártela


═══════════════════════════════════════════════════════════════
  📋 INFORMACIÓN DEL PANEL DE DEBUG
═══════════════════════════════════════════════════════════════

  El panel te dirá:

  Status = DISCONNECTED
    → Deberías ver el botón "Conectar WhatsApp"

  Status = QR_PENDING
    → Deberías ver el QR o "Generando..."

  Status = CONNECTED
    → Deberías ver "WhatsApp conectado exitosamente"

  Status = otro valor
    → Hay un problema, verás un botón "Refrescar Página"
